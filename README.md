# X Keyword Notifier

X Keyword Notifier は、X/Twitter の投稿入力欄に指定した文字列やローマ字読みが含まれたら知らせる Chrome/Edge 拡張機能です。

初期キーワードは `ushi` です。たとえば `うし`、`牛`、`相思相愛` は読みの中に `ushi` が含まれるため検出されます。

## 主な機能

- X/Twitter の投稿・返信入力欄を監視します。
- 本文そのもの、かな読み、ローマ字読みを照合します。
- 例: `相思相愛` は `soushisouai` としても照合されます。
- 任意で INIAD AI MOP によるAI検出を有効化できます。
- API key は利用者が拡張機能のポップアップから入力します。
- キーワードを検出すると、画面内トーストで知らせます。
- 必要に応じてブラウザ通知も表示できます。
- 拡張機能のポップアップから複数のキーワードを変更できます。
- 設定はブラウザ内の拡張機能ストレージに保存します。
- 解析ツール、外部サーバー、リモートスクリプトは使いません。

## 対応ブラウザ

- Google Chrome
- Microsoft Edge
- Manifest V3 に対応した Chromium 系ブラウザ

## インストール

### GitHub Release から入れる場合

1. Releases から `x-keyword-notifier.zip` をダウンロードします。
2. zip を展開します。
3. Chrome または Edge で `chrome://extensions/` を開きます。
4. デベロッパーモードをオンにします。
5. 「パッケージ化されていない拡張機能を読み込む」を押します。
6. 展開したフォルダを選択します。
7. `https://x.com/` を開き、投稿欄に `うし`、`牛`、`相思相愛` などを入力します。

### ソースから入れる場合

1. このリポジトリをダウンロードまたは clone します。
2. Chrome または Edge で `chrome://extensions/` を開きます。
3. デベロッパーモードをオンにします。
4. 「パッケージ化されていない拡張機能を読み込む」を押します。
5. このリポジトリのフォルダを選択します。

## キーワードを変更する

1. ブラウザのツールバーにある拡張機能アイコンを押します。
2. 通知したい文字列またはローマ字読みを入力します。複数ある場合は改行またはカンマで区切ります。
3. 画面内トースト、ブラウザ通知のオン/オフを選びます。

## 読み検出について

この拡張機能は `kuromoji` の辞書を拡張内に同梱し、投稿入力欄の文章を読み仮名化してから `wanakana` でローマ字に変換します。処理はすべてブラウザ内で行われ、外部サーバーには送信しません。

日本語の読みは文脈によって揺れるため、固有名詞や特殊な当て字では辞書と異なる場合があります。

## INIAD AI MOP 検出

ローカル検出で見つからない文章だけ、INIAD AI MOP に問い合わせるハイブリッド検出を使えます。

1. 拡張機能アイコンを押します。
2. `INIAD API key` に自分の API key を入力します。
3. `INIAD AI MOPでも検出` をオンにします。
4. 必要に応じて AI モデルを選びます。

API key は `chrome.storage.local` に保存されます。AI検出をオンにした場合だけ、投稿入力欄の文章が `https://api.openai.iniad.org/api/v1/chat/completions` に送信されます。入力ごとの過剰な送信を避けるため、ローカル検出で見つからない時だけ、入力が止まってから短い待ち時間を置いて問い合わせます。

## 配布用 zip を作る

実行:

```sh
npm run package
```

作成先:

```text
dist/x-keyword-notifier.zip
```

この zip を GitHub Release にアップロードすると、利用者がリポジトリを clone しなくても使えます。

## 権限

この拡張機能は次の権限を使います。

- `storage`: キーワードと通知設定をブラウザに保存します。
- `notifications`: ブラウザ通知を表示します。
- `https://x.com/*` / `https://twitter.com/*`: X/Twitter の投稿入力欄を確認するため、このサイト上だけで動作します。
- `https://api.openai.iniad.org/*`: AI検出を有効にした場合に INIAD AI MOP API へ問い合わせます。

## プライバシー

詳しくは [PRIVACY.md](PRIVACY.md) を読んでください。投稿入力欄の文字列はブラウザ内で確認するだけで、外部には送信しません。

## 開発

初回セットアップ:

```sh
npm install
npm run prepare-vendor
```

拡張機能のファイルをチェック:

```sh
npm run check
```

ビルドは不要です。ファイルを編集したら、`chrome://extensions/` で拡張機能を再読み込みしてください。
